2023年度「Trusted Web の実現に向けたユースケース実証事業」におけるユースケースのうち，株式会社ORPHEによる「下肢運動器疾患患者と医師、研究者間の信用できる歩行データ認証・流通システム」をご紹介します。


事業内容

  • スマートフットウェアによる歩行分析【1】を軸に，変形性膝関節症などの下肢運動器疾患患者，医師，研究者間のデータ流通を可能とするアプリケーション・システムの開発・提供

図: ORPHE FOOTWEAR EASYRUN SHIBUYA 3.0


  • 患者が自身のデータの主体的な管理権限を有しながら，簡易にデータ共有ができるシステムの提供
  • リアルワールドデータを利活用する第三者機関（研究機関/製薬企業など）が，信頼できるデータにアクセスできるシステムの開発
  • 患者が自身のデータを提供することでインセンティブ（ポイントトークンなど）が得られるエコシステムの設計


社会的・経済的な価値

  • 変形性膝関節症は膝関節の軟骨が，加齢や肥満，異常な動作の繰り返しなど様々な原因で変性し，膝関節の変形や痛みを引き起こす疾患であり，本邦における推定患者数は3000万人に上ると言われている【2】．また，変形性関節症治療薬の世界市場規模は2021年で74億ドル，2026年に112億ドル，市場の平均年成長率は8.6%で推移していくと見込まれている【3】．
  • 変形性膝関節症のアセスメントにおいて歩行分析は重要とされているが，特に日常生活内の歩行を効率的に分析し，簡易かつ安全にデータを共有することは困難であった．
  • 本実証事業で開発するシステムは，患者が自身で日常内の歩行分析を行い，より適切な行動/習慣を身につけるとともに，医師や研究機関に定量的な歩行データと主観的な痛みなどのデータを共有することで，効果的な診療・指導や研究開発の促進につながると考えられる．
  • 変形性膝関節症以外の下肢運動器疾患や神経疾患など，歩行分析が関わる疾患分野は多く，今後の横展開が進むことで歩行研究の加速，ひいては健康寿命の延伸につながると考えられる．



Trusted Webの実現により解決する内容

前回の実証実験において開発した，Verifiable credentialや分散型ストレージを活用したデータ流通システムに加え，生体認証による本人のデータであることの信頼性向上や，同意撤回に伴うポイント返戻や患者のインセンティブ設計を充実させることでよりスケール可能なエコシステムの確立を目指す．


実装するシステムアーキテクチャアプリ概要

ユーザが取得したデータはデジタルウォレット（Woollet【4】; DataGateway社製）に紐づけられData credentialが発行されるとともに，暗号化されてORPHECloudを通じてWoolletTEEに送付されたのち、IPFS（InterPlanetary File System）にデータ保存される．リレーションシップクレデンシャルによってパートナーシップ登録されたPartner（医療機関/研究機関など）は，プレサイスターゲティング技術*を用いて患者情報の秘匿性を保ったまま，条件を満たすデータ/ユーザにのみデータ共有リクエストを送信することが可能となる．


*プレサイスターゲティング技術：ゼロ知識証明とVerifiable credentialを用いて，データの機密性を保ったまま任意のデータを持つユーザーを，woolletを通じて発見し，ユーザー同意のもとで当該データへのアクセスをする仕組み（特許申請済：特願2022-146935    ）．本ケースの場合，パートナーが求める一定の条件を満たす対象（例：性別やデータの取得期間など）のみにデータリクエストを送りたい場合にも，患者情報がパートナーには共有されない状態で，データリクエストを送ることができる．データリクエストを受けた患者は，共有をしてもよい項目に対して共有への同意    を行った後に，該当データのみがパートナーに共有される．
